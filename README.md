# Command Line Rabbit Hole
## Welcome To Commmand Line Rabbit Holes

This is a text based game about going through the rabbit holes of the command line fighting monsters collecting various items.
Some items are just treasure item with the sole purpose of being sold. Other items are equipment meant to be equiped to increase defense, damage, and mana for mage classes mostly.
Speaking of Classes the player has the option of 3 different classes : Melee, Ranger, and Mage.
You can play the classes however you want. Melee starts of with higer health and defense. Ranger Starts with less health and defense than Melee but more than Mage. Mage Class is more designed for glass cannon runs.
